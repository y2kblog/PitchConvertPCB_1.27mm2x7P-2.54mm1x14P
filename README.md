# ピッチ変換基板(1.27mm 2×7P⇔2.54mm 1×14P)

## 概要

1.27mmピッチ2×7Pを2.54mmピッチ1×14Pに変換する基板です。

## 仕様
- ピンピッチ：1.27mm 2×7P ⇔ 2.54mm 1×14P
- 基板サイズ：10mm × 35.56mm
- 基板厚：1.6mm
- 基板色：緑
- 表面仕上げ：無鉛はんだ
- 両面スル―ホール
- 固定穴：30mmピッチ, M3 × 2穴
